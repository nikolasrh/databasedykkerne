# Steg 1: Konverter PDFs til tekst

1. Legg tekstbaserte PDF-er i input-mappen.
2. Kjør script: `python main.py`
3. Sjekk output-mappen for markdown-filer.

Har du ikke egne PDF-er? Kjør `./download-example-pdfs.sh` for å laste ned eksempler.

## Frivillige oppgaver

- Ta vare på sidenummer fra PDF-en
- Behold formatering som overskrifter og lister
- Håndter scannede PDF-er med Optical Character Recognition (OCR)
