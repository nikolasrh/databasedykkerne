# Steg 2: Lag tekst-embeddings

1. Kopier markdown fra step-1: `cp ../step-1-convert-pdfs-to-text/output/*.md input/`
2. Kjør script: `python main.py`
3. Sjekk output-mappen for JSON-filer.

- Eksperimenter med chunk-størrelse og overlapp
- Prøv andre embedding-modeller, f.eks. NbAiLab/nb-bert-base som er utviklet av Nasjonalbiblioteket for norsk tekst.
- Prøv å embedde andre ting enn tekst
